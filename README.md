# Covid-19-Data-Aggregation
2020新型冠状病毒数据统计及新闻聚合系统（毕设）

# 相关技术

1. 项目后台使用`Python`语言编写，基于`Flask`框架技术；
2. 前台页面使用`HTML`搭建，结合`jQuery`、`bootstrap`技术；
3. 爬虫主要基于`requests`模块；
4. 数据存储在`MongoDB`数据库中

# 主要功能

1. 用户

    ① 疫情数据实时统计展示

    ② 辟谣信息/疫情新闻/复工新闻/援助信息等数据的爬取与聚合

    ③ 讨论区（话题的发布与评论）

2. 管理员

    ① 用户管理

    ② 讨论区话题管理

    ③ 评论管理
